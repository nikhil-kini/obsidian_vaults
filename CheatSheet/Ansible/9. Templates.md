```
ansible_project
 |__roles
	|__base
	   |__files
	   |__tasks
	   |__templates
```

- Make `templates` directory under role
- Copy `/etc/ssh/sshd_config` inside `templates`
```sh
cp /etc/ssh/sshd_config sshd_config_ubuntu.j2
```
- To grab remotely from server
```sh
scp ip_address:/etc/ssh/sshd_config sshd_config_centos.j2
```
- edit `ssh.j2` file and add
```
AllowUsers {{ssh_user}}
```
-  add variables for each file under `host_vars/`
```yml
ssh_users: "nikhil ansible"
ssh_template_file: ssh.config_ubuntu.j2 # file name for sshd config
```
- add task to `base` role
```yml
 - name: openssh | generate sshd_config file from template
   tags: ssh
   template:
     src: "Template:Ssh template file"
     dest: /etc/ssh/sshd_config
     owner: root
     group: root
     mode: 0644
   notify: restart_sshd
```
- Make directory under role `roles/base`
```sh
 mkdir handlers
 cd handlers
```
- Make `main.yml`
```yml
 - name: restart_apache
   service:
     name: sshd
     state: restarted
```
- Run playbook
