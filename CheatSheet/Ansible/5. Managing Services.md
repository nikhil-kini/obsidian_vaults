
```yml
 - hosts: web_servers
   become: true
   tasks:
 
   - name: install httpd package (CentOS)
     tags: apache,centos,httpd
     dnf:
       name:
         - httpd
         - php
       state: latest
     when: ansible_distribution == "CentOS"
 
   - name: start and enable httpd (CentOS)
     tags: apache,centos,httpd
     service:                              
       name: httpd                         # name of the service to be strated
       state: started                      # `systemctl start httpd` command
       enabled: yes                        # `systemctl enable httpd` command
     when: ansible_distribution == "CentOS"

   - name: change e-mail address for admin
     tags: apache,centos,httpd
     lineinfile:                         # module to edit a line in a file
       path: /etc/httpd/conf/httpd.conf  # path of file to be edited
       regexp: '^ServerAdmin'            # regex to find the line to edit
       line: ServerAdmin somebody@somewhere.net  # replace entire linefile with this
     when: ansible_distribution == "CentOS"
     register: httpd_any                   # Register the task to a varilable
 
   - name: restart httpd (CentOS)
     tags: apache,centos,httpd
     service:
       name: httpd
       state: restarted
     when: httpd_any.changed      # Boolean, if changed is = true
 
```

- Be careful when using `lineinfile` has it can make unexpected changes
- To verify run *playbook twice, note nothing has changed to confirm state is the same in second run.*
- Note: One `register` variable for one task.
