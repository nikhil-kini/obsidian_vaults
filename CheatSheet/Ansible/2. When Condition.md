- Task executes **if the condition is met in the when block.**
- You **can use ansible variables** from `ansible all -m gather_facts --limit ip_address` 
- Below example is a demo, code can be modified for efficiency.
```yml
---

- hosts: all
  become: true
  tasks:

  - name: update repository index for Ubuntu
    apt:
      update_cache: yes
    when: ansible_distribution_version in ["Ubuntu","Debian"] 

  - name: install apache2 package for Ubuntu
    apt:
      name: apache2
      state: latest
    when: ansible_distribution == "Ubuntu" and ansible_distribution_version == "20.04"

  - name: add php support for apache for Ubuntu
    apt:
      name: libapache2-mod-php
      state: latest
    when: ansible_distribution == "Ubuntu"

  - name: update repository index for CentOS
    dnf:
      update_cache: yes
    when: ansible_distribution_version == "CentOS"

  - name: install php package for CentOS
    dnf:
      name: httpd
      state: latest
    when: ansible_distribution == "CentOS"

  - name: add php support for apache for CentOS
    dnf:
      name: php
      state: latest
    when: ansible_distribution == "CentOS"
```

### Condensed and efficient way
```yml
 ---
 
 - hosts: all
   become: true
   tasks:
 
   - name: install apache and php
     package:
       name:
         - "{{apache_package}}"
         - "{{php_package}}"
       state: latest
       update_cache: yes
```

`inventory` file must be modified, to include host variables.
```
172.16.250.132 apache_package=apache2 php_package=libapache2-mod-php
172.16.250.133 apache_package=apache2 php_package=libapache2-mod-php
```
