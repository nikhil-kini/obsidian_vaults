## Introduction

Install apache and more. `install_apache.yml`
```yml
 ---
 
 - hosts: all                      # on all connected device
   become: true                    # become sudo user
   tasks:                          # perform task
 
   - name: update repository index # name of task 1 can be any
	 apt:                          # package manger in connected device
       update_cache: yes           # update package manager 'apt update'
 
   - name: install apache2 package # name of task 2 can be any
     apt:
       name: apache2               # name of package to be installed
       state: latest               # version of package to be installed
 
   - name: add php support for apache
     apt:
       name: libapache2-mod-php
       state: latest
```

Remove apache and more. `remove_apache.yml`
```yml
 ---
 
 - hosts: all
   become: true
   tasks:
 
   - name: remove apache2 package
     apt:
       name: apache2
       state: absent          # makes sure state of machine dosn't have apache
 
   - name: remove php support for apache
     apt:
       name: libapache2-mod-php
       state: absent
```

**To run Playbook**
```sh
 ansible-playbook --ask-become-pass path_to_file.yml
```

ex:
```sh
 ansible-playbook --ask-become-pass install_apache.yml
```
