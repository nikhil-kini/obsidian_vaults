#obsidian 

## Installation
Use **Flatpack** package manager for installation in the Linux distros, for more options visit [install](https://obsidian.md/)

## Usage
Uses [[Markdown]] for text formatting. With few additional functionalities.

### Terminologies
**Vault** - Folder which contains the notes.

### Shortcuts
`Ctrl + p` - for the command prompt.
`Ctrl + Click` - to go inside a link, in edit
`Ctrl + e`  - to toggle edit and read only mode
`Ctrl + f` - to find 
`Ctrl + Alt + <-` - to move to previous page
`Ctrl + Alt + ->` - to move forward
`Ctrl + Click_on_Note` - to open note in a new pane
`Ctrl + n` - for new note
`Ctrl + o` - for quick menu

[refer this for more](https://forum.obsidian.md/t/obsidian-hotkeys-favorites-and-best-practices/12125)

### Setting Changes
*Editor* - turn on spell check
*Files and Link* - turn on Automatic internal link update
